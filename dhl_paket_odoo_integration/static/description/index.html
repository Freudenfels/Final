DHL Paket(Business Customer API) Shipping Integrations


Our DHL Paket(Business Customer API) Shipping Integration will help you connect with DHL Paket(Business Customer API) with Odoo. Submit order information from Odoo to DHL Paket(Business Customer API) and Get multiple shipping labels, Cancel shipping label and Order Tracking number from Odoo to DHL Paket(Business Customer API).

Key Features

  • Configure DHL Paket(Business Customer API) API credentials in odoo
  • Configure number of shipping delivery methods in odoo
  • Export Delivery Order in DHL Paket(Business Customer API) while validate delivery order in Odoo.
  • Get multiple Shipping labels from DHL Paket(Business Customer API) to Odoo and Attach in the Delivery Order.
  • Save time and prevent human error by creating labels automatically with your favorite shipping service, saved package information, and new faster workflow.
  • Get Order Tracking number from carriers to Odoo.
  • It is possible to Cancel shipment from odoo to DHL Paket(Business Customer API).
  • By click on tracking button in delivery order, Dynamically redirect to DHL Paket(Business Customer API) and track shipment and get order status.
