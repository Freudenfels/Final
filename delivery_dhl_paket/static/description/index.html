Delivery Method - DHL Paket(Business Customer Shipping Api Integration)

Inventory > Configuration > Delivery > Delivery Method

  • Service: BulkyGoods for managing bulky products. Configure in packages.
  • Added DHL product Warenpost



Installation




  • Install the suds python package: suds-community
  • sudo pip3 install suds-community

Configuration





Create a new delivery method and specify the delivery type as DHL DE Paket and specify the other DHL DE Paket configuration details. As DHL DE Paket webservice is not providing any shipping cost, you can specify a fixed shipping cost.
